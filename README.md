# cub3D

cub3D est un projet du cursus 42.
Le but est de créer un **jeu en 3D type Wolfenstein**, avec un moteur de rendu simple basé sur le raycasting.

## Compilation
 
```bash
make
```

## Utilisation

```bash
./cub3D map.cub
```

## Contenu

* Affichage 3D de cartes en utilisant le **raycasting**
* Gestion des textures et des sprites
* Déplacement du joueur, collisions et orientation de la caméra
